##  Descripción  

En este trabajo práctico se analiza el rendimiento de sistemas de cómputo desde dos enfoques principales:

1. **Evaluación mediante benchmarks**: selección y análisis de herramientas de medición para representar tareas reales de uso cotidiano.
2. **Análisis de performance de código**: utilización de técnicas de *time profiling* para estudiar el tiempo de ejecución de funciones y detectar cuellos de botella.

El objetivo es comprender cómo distintos componentes del sistema (CPU, memoria, almacenamiento y GPU) influyen en el desempeño general y cómo medirlo de manera adecuada.

---

##  Objetivos  

- Identificar métricas relevantes de rendimiento.
- Seleccionar benchmarks adecuados según el tipo de tarea.
- Comparar el desempeño de distintos procesadores en escenarios reales.
- Aplicar herramientas de profiling para analizar programas.
- Interpretar resultados y extraer conclusiones fundamentadas.
